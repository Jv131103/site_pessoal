soma_binaria.py 
def soma_binaria(A, B):
    n = len(A)
    C = [0] * (n + 1)
    carry = 0

    for i in range(n - 1, -1, -1):
        soma = A[i] + B[i] + carry

        if soma == 0:
            C[i + 1] = 0
            carry = 0
        elif soma == 1:
            C[i + 1] = 1
            carry = 0
        elif soma == 2:
            C[i + 1] = 0
            carry = 1
        else:
            C[i + 1] = 1
            carry = 1

    C[0] = carry
    return C

A = [1, 0, 1, 1]
B = [1, 1, 0, 1]

print(soma_binaria(A, B))

Explicação

  • Objetivo: somar dois números binários representados por listas.
  • Ideia: percorre os bits da direita para a esquerda.
  • Carry: guarda o “vai um” da soma.
  • Resultado: retorna uma nova lista com um espaço extra no início.
busca_binaria.py 
def busca_binaria(lista, alvo):
    inicio = 0
    fim = len(lista) - 1

    while inicio <= fim:
        meio = (inicio + fim) // 2

        if lista[meio] == alvo:
            return meio
        elif lista[meio] < alvo:
            inicio = meio + 1
        else:
            fim = meio - 1

    return -1

numeros = [1, 3, 5, 7, 9, 11, 13]
print(busca_binaria(numeros, 7))

Explicação

  • Objetivo: encontrar um valor dentro de uma lista ordenada.
  • Estratégia: divide a área de busca pela metade.
  • Vantagem: é muito mais rápida que procurar item por item.
  • Retorno: índice do valor ou -1 se não encontrar.
validador_email_regex.py 
import re


def validar_email(email):
    padrao = r"^[\w\.-]+@[\w\.-]+\.\w{2,}$"
    return bool(re.match(padrao, email))

emails = [
    "joao@email.com",
    "email_invalido",
    "contato@empresa.com.br"
]

for email in emails:
    print(email, validar_email(email))

Explicação

  • Objetivo: validar se uma string parece um e-mail.
  • Regex: procura usuário, arroba, domínio e extensão.
  • Uso real: pode ajudar em formulários antes de salvar dados.
consumo_api_requests.py 
import requests


def buscar_usuario_github(usuario):
    url = f"https://api.github.com/users/{usuario}"
    resposta = requests.get(url, timeout=10)

    if resposta.status_code == 200:
        dados = resposta.json()
        return {
            "nome": dados.get("name"),
            "bio": dados.get("bio"),
            "repositorios": dados.get("public_repos")
        }

    return {"erro": "Usuário não encontrado"}

print(buscar_usuario_github("Jv131103"))

Explicação

  • Objetivo: consumir uma API pública com Python.
  • Requests: faz a requisição HTTP.
  • JSON: transforma a resposta em dicionário Python.
  • Timeout: evita que o programa fique travado.
calculadora_imc.py 
def calcular_imc(peso, altura):
    imc = peso / (altura ** 2)

    if imc < 18.5:
        classificacao = "Abaixo do peso"
    elif imc < 25:
        classificacao = "Peso normal"
    elif imc < 30:
        classificacao = "Sobrepeso"
    else:
        classificacao = "Obesidade"

    return round(imc, 2), classificacao

resultado = calcular_imc(70, 1.75)
print(resultado)

Explicação

  • Objetivo: calcular o IMC com peso e altura.
  • Condição: classifica o resultado por faixas.
  • Retorno: entrega o valor arredondado e a classificação.
